Fun Travels Frontend-Only Project

Files:
- index.html
- results.html
- payment.html
- success.html
- mybookings.html
- loader.js

How to run:
1. Put all files in the same folder.
2. Open index.html in your browser.
3. Search route, select seats, continue to payment, and book.

Storage keys used:
- ft_search
- ft_payment_draft
- ft_bookings
- ft_takenSeats
- ft_last_paid
- ft_emails
